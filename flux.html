Flux in react

  1. Created by facebook
  2. Flux is a pattern for UI which runs on a unidirectional flow
  3. has a centralized dispatcher
  4. It manages the data flow
  5. The primary difference of Flux vs Redux is that Flux includes multiple Stores per app, but Redux includes a single Store per app.
  6. Flux is the Frontend architecture that Facebook uses for building client-side web applications.
